GitHub Setup

To upload adventures, you need a GitHub Personal Access Token with 'repo' permissions.

Click here to create a token (select 'repo' scope)

How to Add an Adventure

  1. Set up your GitHub token above (one-time setup)
  2. Fill out all the fields below with your adventure details
  3. Upload your GPX file from onX
  4. Upload photos and select a cover photo
  5. Click "Upload Adventure to GitHub" - everything will be committed automatically!
Use lowercase letters, numbers, and hyphens only
